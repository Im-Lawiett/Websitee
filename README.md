# AI Collection — RianModss

Koleksi website AI terbaik: ChatGPT, Claude, Gemini, Grok, DeepSeek, Perplexity, Dola, dan lebih banyak lagi. Temukan dan simpan prompt terbaik kamu.

## Tech Stack

- **Next.js 14** — App Router, SSR/SSG
- **React 18** — UI components
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling
- **Framer Motion** — Animations
- **GitHub API** — Prompt data storage

## Features

- 🤖 **12+ AI Tools** — ChatGPT, Claude, Gemini, Grok, DeepSeek, Perplexity, Dola, Copilot, Midjourney, Llama, Qwen, Stable Diffusion
- 📚 **Prompt Library** — Simpan, cari, edit, dan salin prompt
- 🔌 **GitHub Sync** — Data prompt tersimpan di GitHub repo kamu
- 🔒 **Password Protection** — Admin area dilindungi password
- 📱 **Responsive** — Mobile-first design
- ⚡ **Fast** — Next.js optimized, Vercel ready

## Deploy ke Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/websitee)

1. Fork/clone repo ini
2. Import ke [Vercel](https://vercel.com)
3. Deploy otomatis!

## Setup Lokal

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000)

## GitHub Integration

Di website, klik "Setup Server" dan masukkan:
- GitHub Personal Access Token (dengan izin `repo`)
- Username GitHub kamu
- Nama repository untuk menyimpan data
